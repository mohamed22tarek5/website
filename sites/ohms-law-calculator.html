Circuit Analysis Formulas

  • Series Circuit: Rtotal = R₁ + R₂ + R₃ + ...
  • Parallel Circuit: 1/Rtotal = 1/R₁ + 1/R₂ + 1/R₃ + ...
  • Current (Series): I = V / Rtotal (same through all components)
  • Current (Parallel): Itotal = I₁ + I₂ + I₃ + ...
  • Voltage Drop (Series): Vn = I × Rn
  • Power Dissipation: P = I² × R = V² / R
