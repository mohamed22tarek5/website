Circuit Analysis Formulas

  • SERIES Circuit: Rtotal = R\u2081 + R\u2082 + R\u2083 + ...
  • PARALLEL Circuit: 1/Rtotal = 1/R\u2081 + 1/R\u2082 + 1/R\u2083 + ...
  • Current (Series): I = V / Rtotal (same through all components)
  • Current (Parallel): Itotal = I\u2081 + I\u2082 + I\u2083 + ...
  • Voltage Drop (Series): Vn = I \u00D7 Rn
  • Power Dissipation: P = I\u00B2 \u00D7 R = V\u00B2 / R
