Electrical Engineering Formulas Reference

Basic Formulas

  • Ohm's Law: V = I x R
  • Power: P = V x I
  • Energy: E = P x t
  • Charge: Q = I x t
  • Resistance: R = rho x (L/A)

Circuit Formulas

  • Series: R_total = Sum(R_i)
  • Parallel: 1/R_total = Sum(1/R_i)
  • Voltage Divider: V_out = V_in x (R2/(R1+R2))
  • Current Divider: I_1 = I_total x (R_total/R_1)

Power Formulas

  • AC Power: P = V x I x PF
  • Three-Phase: P = sqrt(3) x V_L x I_L x PF
  • Reactive Power: Q = V x I x sin(phi)
  • Apparent Power: S = V x I
  • Power Factor: PF = cos(phi)
